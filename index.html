Dynamic Javascript Quiz

Hello! I have created this dynamic quiz using javascript to practice interview questions. The only problem is that you are running against time! Once you click the "Start Quiz" button, you will be taken to the first question and the clock will start ticking! Clock will start at 30 seconds and will count down to zero. The less time it takes you to answer all 4 questions, the higher you will score in the leadersboard. Oh yeah... for every wrong answer, 10 seconds will be deducted from the clock. Ready, Set, Go!
